# Weather App L+L

Simple web app that reveals weather and temperature at your current location, based on IP. 

## Start project

1. npm install
2. npm run dev


### Stack 

* React 17.0.2
* Tailwind CSS
* Boilerplate by [Vite](https://vitejs.dev/)
* Weather data and geolocation by [WeatherApi](https://www.weatherapi.com/)
